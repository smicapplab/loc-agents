# Personal AI Agents Collection

A repository for autonomous AI agents designed to automate personal workflows, research, and productivity tasks.

## Agents

### [PH Job-Seek Agent](./job-seek)
An autonomous agent that scrapes job boards (LinkedIn, Indeed, Seek, RemoteOK, WWR), filters them against a personal resume using Gemini 2.5 Flash, and sends a curated daily email of the best matches.

---
*Created with Gemini CLI.*
